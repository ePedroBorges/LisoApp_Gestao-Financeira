# LisoApp - Gestao Financeira para Lisos

Link para acesso: https://liso-app-44345bea.base44.app

## Introdução:

O LisoApp é um aplicativo de gestão financeira desenvolvido com foco total na simplicidade. A proposta é entregar exatamente o que o usuário precisa para controlar seu dinheiro de forma clara, rápida e sem complicações. Nada de telas confusas, excesso de funções ou bloqueios: apenas o básico funcionando muito bem — como deve ser.

O LisoApp nasceu para ajudar pessoas comuns a assumirem o controle da própria vida financeira, especialmente quem vive no modo “liso”, mas quer evoluir.

## Objetivo do Aplicativo:

Oferecer uma ferramenta leve, intuitiva e eficiente para registrar entradas, saídas, metas financeiras e gastos mensais, permitindo que o usuário acompanhe sua jornada rumo ao equilíbrio financeiro.

## Funcionalidades Principais:

1. Criação de Conta e Login:
- Cadastro simples e rápido.
- Login seguro para sincronizar dados.
- Informações básicas de perfil: situação de emprego, salário, empresa, escolaridade e tema do app.

2. Controle de Transações:
- Registre todas as movimentações financeiras:
Entradas;Valor; Data;Origem (salário, freelas, benefícios etc.); Saídas; Valor; Data; Destino (conta de luz, mercado, internet etc.); Categoria (alimentação, transporte, lazer, outros).

3. Relatórios com Filtros:
- Visualize suas transações com rapidez: Filtro por tipo (entrada, saída ou ambos); Filtro por mês; Filtro por ano; Exibição instantânea das transações recentes e valores totais.

4. Gestão de Objetivos Financeiros:
- Defina e acompanhe metas:Título e descrição; Valor alvo; Valor atual; Prazo; Painel dedicado para visualizar o progresso de cada objetivo.

5. Gestão de Gastos Fixos e Variáveis:
- Organize suas despesas mensais:
Gastos Fixos; Mensalidades recorrentes (internet, aluguel, streaming etc.); Gastos Variáveis; Custos não fixos (lazer, compras esporádicas, transporte, etc.)

- Ambos com: Descrição; Valor; Vencimento; Lista detalhada e totalizador automático

6. Consultor Financeiro Integrado (IA):
- Um chatbot interno que auxilia em perguntas como: “Como posso economizar mais?”; “Analise meus gastos.”; “Dicas para investir.”; “Como fazer um orçamento?”; Tudo dentro do próprio aplicativo.

7. Área do Usuário:
- Edição de informações pessoais.
- Ajuste de tema (escuro/claro).
- Configurações gerais.

8. Página “Sobre”:
- Informações sobre a AnchorLab.
- Possibilidade de contribuir com o projeto via PIX.
- Links de contato (LinkedIn e WhatsApp).


Tecnologias Utilizadas:
- Lovable (Vibecoding)
- JavaScript
- HTML/CSS
- APIs e automações
- Integração com IA
